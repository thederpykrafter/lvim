# lvim keys

## LazyGit

| key | action |
| ? | Keybinds Menu |
| P | Push |
| p | Pull |
| c | Checkout |
| C | Commit using git editor |
| m | Merge/Rebase options |
| a | Stage all files |
| i | Ignore file |
| <Enter> | Open file or expand/collapse directory |
| ` | Toggle file tree view |
| R | Refresh |
| Z | Undo |
| <C-z> | Redo |
| e | Edit |
| o | Open file |
